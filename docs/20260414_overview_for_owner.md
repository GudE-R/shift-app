# ShiftCraft ― オーナー向けご紹介資料

**Local-first AI シフト管理アプリ**
2026-04-14

---

## 1. ShiftCraft とは

ShiftCraft は、**店舗オーナーの手元 PC だけで動く**シフト作成アプリです。
スタッフの個人情報をクラウドに一切アップロードせず、AI が条件に沿ったシフトの叩き台を自動生成します。最終判断は常にオーナーの手に残ります。

> 「AI は叩き台を作る。決めるのはオーナー。」

### 3つの特長

| | 特長 | 内容 |
|---|------|------|
| 01 | **ローカル完結で安心** | スタッフ名・シフト情報は手元の SQLite にのみ保存。インターネットに流れません。 |
| 02 | **AI が下書きを数秒で作成** | 曜日可否・NG日・夜勤可否・ポジション適性を考慮し、労働時間が偏らないよう自動配置。 |
| 03 | **警告ベースで柔軟に運用** | 休憩不足や二重配置は赤/黄で警告。禁止ではなく気づかせるので、現場都合の例外も通せます。 |

---

## 2. こんな悩みを解決します

- 毎月のシフト作成に **半日〜1日**かかる
- スタッフの **希望・NG日**を Excel で管理しきれない
- 休憩時間や連続勤務など **労基ルール**の抜け漏れが怖い
- クラウド型シフトアプリに **個人情報を預けるのが不安**
- 印刷して貼り出す用の **見やすい表**を別途作り直している

ShiftCraft はこの 5 つを 1 アプリで解消します。

---

## 3. 使い方の流れ（5ステップ）

### Step 1. 店舗・ポジションを登録
店舗名、カラー、営業時間を登録。レジ／キッチンなどのポジションを設定します。

### Step 2. スタッフを登録
表示名、曜日別の出勤可否（○/△/×）、絶対NG日、夜勤可否、対応できるポジションを入力。

### Step 3. 必要人員を設定
「火曜のキッチン 10:00〜14:00 は 2 人」のように、ポジション×時間帯ごとの必要人数を入れます。

### Step 4. AI でシフトを自動生成
ボタンひとつで 1 ヶ月分を一括生成。スタッフの希望・制約を尊重したうえで、労働時間ができるだけ均等になるよう配置します。

### Step 5. 手動で微調整 → PDF / CSV 出力
- カレンダー上でドラッグ＆ドロップで入れ替え
- 手動編集したコマは保護され、**AI 再生成で上書きされません**
- A3 横レイアウトの **PDF**、Excel 互換の **CSV** でワンクリック出力

## 4. 主な画面

| 画面 | 役割 |
|------|------|
| **ダッシュボード** | 週間サマリー・充足率・警告数を一目で把握 |
| **シフトカレンダー** | スタッフ×日付のグリッド。月間／週間を切替、色で店舗を区別 |
| **スタッフ管理** | 出勤条件・NG日・対応ポジションをまとめて編集 |
| **必要人員設定** | 日付×ポジション×時間帯の要員数をテーブルで設定 |
| **PDF / CSV 出力** | 印刷用 A3 シフト表、Excel 取り込み用 CSV |

## 5. リアルタイム警告（労基準拠）

作業中、常時バックグラウンドでチェックし、問題があれば色で知らせます。

| 内容 | 判定基準 | 表示 |
|------|---------|------|
| 休憩不足 | 6h 超 → 45分 / 8h 超 → 60分（労基法第34条） | 赤 |
| 二重配置 | 同一スタッフが同時刻に複数店舗 | 赤（ブロック） |
| 長時間労働 | 週 40h 超 / 日 12h 超 | 黄 |
| 連続勤務 | 既定日数（初期値 5 日）を超過 | 黄 |

**警告は出しますが、最終的に通すかはオーナーが決められます。**

---

## 6. 安心ポイント ― なぜ「ローカル完結」なのか

- スタッフの本名・連絡先は **この PC の中だけ**。退職後のデータ削除も一発。
- AI（Gemini）に渡すのは `staff_001` のような **匿名 ID のパズル**のみ。名前は送られません。
- インターネットが止まっても **全機能が動作**します。
- 将来のクラウド同期（Phase 2）も **端末側で暗号化してから**送る設計。サーバー運営者も中身を見られません。

---

## 7. 動作環境

- **対応 OS**: Windows / macOS / Linux（デスクトップアプリ）
- **推奨**: メモリ 4GB 以上、ディスク空き 200MB 以上
- **ネット接続**: 初回ダウンロード時のみ必須。通常運用はオフラインで OK
- **インストール**: インストーラをダブルクリック → 起動 → パスワード設定で完了
